------Breve descripción sobre el programa---- 
El programa aplica la idea de programación orientada a objetos mediante el uso de figuras geométricas. La clase base FiguraGeometrica establece atributos privados (_ancho, _alto) utilizando encapsulamiento a través de @property y @setter para verificar que los valores sean superiores a cero. Las clases Rectángulo, Cuadrado y Circunferencia son subclases de la clase base y reescriben los métodos area() y perimetro() utilizando sus propias fórmulas. Las funciones sumar_areas() y sumar_perimetros() del programa muestran polimorfismo; estas dos funciones aceptan una lista de figuras y calculan las sumas totales, sin considerar la clase particular de cada figura. Esto posibilita que el código sea versátil y reutilizable, puesto que cualquier figura que herede de FiguraGeometrica tiene la capacidad de emplear estas funciones de manera automática.
